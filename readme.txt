A Bitlove Widget

== Description ==

This twitter widget add a Bitlove Button to your page with a link to your Profile Page or your Feed Page.


== Installation ==

1. Upload the entire folder `troll-bitlove-widget` to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
